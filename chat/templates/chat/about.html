What:

Fluent friend is a chatbot hooked up to ChatGPT 3.5 which both responds to messages and creates corrections for them. Then a little magic behind the scenes takes those corrections and displays them with the changes highlighted.

Why:

It's hard to find a friend, let alone a fluent one, who speaks the language you're learning. Well, it's the 21st century and we don't need friends anymore! Instead, we can talk to chat bots (the end is nigh!).

Fluent friend is based on well studied and simple methods to help you learn. Essentially, practicing a skill helps reify it in your little brain. But corrections are necessary to make sure you are learning accurately. Recall -> corrections -> repeat. That's the name of the game!

That's the method the creator of fluent friend was using to practice their French! Although it was obvious that ChatGPT was not equiped to help learn in that way. So they decided test out their skills and create a website for all to use.

Who:

This web app was created by Kyle Kent, a passionate learner of all things French and tech. Although they're a data engineer by trade, they wanted to take a stab at creating a website. Et voila!
